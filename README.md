# TikTok_invite
## 功能

- 获取达人数据
- 分段爬取
- 数据保存
- 更新达人数据
- 批量上传达人
- 添加标签
- 发送信息
- 删除标签
  

## **数据的获取**

自动筛选关键的数据，通过下滑来获取更多的数据，每次获取12条。

指定每次获取数据的条数，为一轮。

输入总共需要获取多少条有效数据。

每获取一轮数据，将保存到excel表格当中，每次会进行数据的去重操作。

去重后的数据为有效数据。


## 自动添加标签

标签栏里需要有原始标签数值。

自动获取最上层标签数值。

将最上层标签数值+1创建为最新标签。


## 自动上传达人

将文件切割，每一份表格带有20条数据。

分割出来的数据自动上传，添加最新的标签，方便筛选。


## 自动发送邀约

每次选取20位达人，按邀请步骤发送邀请。

自动填写邀请信息、选择商品、设置佣金、发送邀请。


## 自动删除标签
发送成功后返回标签页。

删除发送过消息的达人标签，方便下一次添加。
